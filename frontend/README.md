# EELU Library Hub - مشروع تعليمي كامل ✅

## 📋 ملخص المشروع
مشروع مكتبة رقمية متكامل مع HTML, CSS, و JavaScript - تم بناؤه لأغراض تعليمية.

## ✨ المميزات المنجزة

### 🔐 نظام المصادقة
✅ **Login Page** - تسجيل دخول بالبريد والكلمة المرور
✅ **Register Page** - تسجيل حساب جديد مع التحقق من البيانات
✅ **Session Management** - استخدام localStorage لإدارة الجلسات
✅ **Demo Mode** - يسمح بالدخول بأي بيانات في المرة الأولى

### 🎨 الصفحات الرئيسية
✅ **Combined Page (Home)** - صفحة رئيسية متكاملة مع:
   - Hero Section
   - Dashboard سريع
   - Library Services
   - About Section
   - Statistics
   - Footer

✅ **Search Page** - بحث متقدم مع:
   - البحث الفوري عن الكتب
   - تصفية حسب الكلية (BA/IT)
   - تصفية حسب السنة الدراسية
   - تصفية حسب الفئة
   - عرض عدد النتائج

✅ **My Borrowed Books** - صفحة الكتب المستعارة
   - إحصائيات المستعيرات
   - حالة الكتب (مستعار/محجوز/قريب الانتهاء)

✅ **Accessibility Page** - ميزات إمكانية الوصول
   - التحكم بحجم الخط
   - الوضع عالي التباين
   - قارئ الشاشة

✅ **Contact & Support** - صفحة الاتصال
   - نموذج اتصال كامل
   - معلومات الدعم
   - ساعات العمل

✅ **Book Details Page** - صفحة تفاصيل الكتاب
   - صورة الكتاب
   - معلومات الكتاب
   - أزرار الاستعارة والحجز
   - تحميل PDF

### 💻 ملفات JavaScript المنظمة
- `login.js` - منطق تسجيل الدخول
- `register.js` - منطق التسجيل الجديد
- `combined.js` - منطق الصفحة الرئيسية
- `search.js` - منطق البحث والتصفية
- `my-borrowed.js` - إدارة الكتب المستعارة
- `contact.js` - معالجة نموذج الاتصال
- `accessibility.js` - ميزات إمكانية الوصول
- `book-details.js` - تفاصيل الكتاب

### 🎯 الميزات الوظيفية
✅ تحقق من صيغة البريد الإلكتروني
✅ التحقق من طول كلمة المرور (6 أحرف على الأقل)
✅ حماية الصفحات - إعادة التوجيه إلى تسجيل الدخول إذا لم يتم المصادقة
✅ تحديث بيانات المستخدم في جميع الصفحات
✅ تسجيل الخروج من أي صفحة
✅ البحث والتصفية الفورية للكتب
✅ الروابط والأزرار تعمل بشكل صحيح في جميع الصفحات

## 🚀 كيفية الاستخدام

### البدء الأول
1. افتح `index.html` في المتصفح
2. سوف ترى قائمة الصفحات المتاحة

### تسجيل الدخول
1. انقر على "Login"
2. أدخل أي بريد إلكتروني وكلمة مرور
3. في المرة الأولى، سيتم إنشاء حساب تلقائياً (demo mode)
4. انقر على "Login" للدخول

### استكشاف الميزات
- ابحث عن الكتب في "Search Books"
- تحقق من الكتب المستعارة في "My Books"
- اكتشف ميزات إمكانية الوصول في "Accessibility"
- تواصل معنا من خلال "Contact & Support"

## 📝 ملاحظات تقنية

### localStorage
يتم حفظ بيانات المستخدم بشكل محلي:
- `userName` - اسم المستخدم
- `userEmail` - البريد الإلكتروني
- `userPassword` - كلمة المرور
- `faculty` - الكلية
- `academicYear` - السنة الدراسية
- `isLoggedIn` - حالة تسجيل الدخول

### الأمان التعليمي
⚠️ ملاحظة: هذا مشروع تعليمي فقط
- لا تستخدم localStorage لحفظ كلمات المرور في التطبيقات الحقيقية
- كلمات المرور يجب أن تشفر على الخادم
- يجب استخدام JWT أو Sessions للمصادقة الحقيقية
